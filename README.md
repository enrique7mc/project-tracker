# Project Tracker

Minimal kanban board for tracking projects, goals, and todos. Single page web app with no dependencies.

## Features

- Kanban columns: Backlog, In Progress, Done
- Cards with title, description, project label, and deadline
- Drag and drop between columns
- Filter by project
- Deadline tracking with color-coded warnings
- JSON export/import for backup
- Dark theme
- Data stored in localStorage

## Usage

Open `index.html` in your browser. That's it.

## Data

All data lives in your browser's localStorage. Use Export/Import to backup or transfer data between browsers.
